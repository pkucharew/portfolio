StickerYou Sticker Editor

Sticker Design Application

StickerYou tasked me with developing a front-end template for their custom sticker editor which would work for both desktop & mobile.

I was responsible for developing a front-end which matched their design pixel for pixel and designing user interactions for the application.

Bucketlist Rewards

Angular Application

As front-end developer & themer at Bucketlist I worked within the dev team to develop an employee recognition & rewards software built with Angular on the Django platform.

Bucketlist Rewards is a private social network for businesses to manage employee rewards through a point system which can be redeemed for real world experiences. I contributed to creating the front-end for features like the user's timeline, point leaderboard and events system.

Software as a Service

Angular, Bootstrap, HTML, SASS, Git, Agile methodology.

About Me

Hello, my name is Pauline! 👋
